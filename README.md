# Volatility, Liquidity & Derivatives Risk Analysis

## Overview

This project develops a derivatives risk analytics framework using Python and real NIFTY 50 market data.

## Features

- Market data acquisition
- Volatility estimation
- Liquidity analysis
- Black-Scholes option pricing
- Greeks computation
- Delta hedging
- Value-at-Risk
- Stress testing

## Technologies

- Python
- Pandas
- NumPy
- Statsmodels
- SciPy
- Matplotlib
- yfinance
